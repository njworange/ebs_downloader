# EBS Downloader Plugin

FlaskFarm용 EBS 애니키즈(https://anikids.ebs.co.kr) 자동 수집/다운로드 플러그인입니다.

## 주요 기능

- EBS 애니키즈 프로그램/에피소드 자동 수집
- URL/코드 직접 분석 후 선택 다운로드
- 다운로드는 FlaskFarm `ffmpeg` 플러그인(SupportFfmpeg) 경유
- 전체 길이 우선 다운로드(로그인 쿠키 기반)
- 화질 선택(M50/M20/M10/M05)
- 프리뷰(180초) 차단 옵션
- 자막 다운로드 옵션
- 화이트리스트/블랙리스트 필터
- EBS 계정 로그인으로 쿠키 자동 생성
- 브라우저 세션(Chrome/Edge/Firefox 등)에서 쿠키 직접 읽기
- 쿠키 만료 시 자동 갱신(저장된 계정 또는 브라우저 쿠키)
- 쿠키 파일(cookies.txt / Cookie 문자열)에서 쿠키 불러오기
- 파일명 형식: `{프로그램제목}.E##.yymmdd.1080p-EBS.mp4`

## 기본 설정

1. `설정 > 로그인 쿠키(필수)`에 쿠키를 직접 붙여넣거나
2. `EBS 아이디(임시 입력)`, `EBS 비밀번호(임시 입력)` 입력 후 `계정으로 쿠키 자동 가져오기` 실행
3. 또는 `브라우저 세션 쿠키 가져오기`로 브라우저 로그인 쿠키를 직접 읽어 저장
4. 또는 `쿠키 파일 경로`를 설정하고 `쿠키 파일에서 불러오기`로 쿠키를 읽어 저장
5. `쿠키 만료 시 자동 갱신`을 `On`으로 두면 만료 시 자동 재로그인 시도
6. 저장 경로/화질/필터 설정 후 저장

## URL/코드 직접 다운로드(선택)

`자동 > 선택` 메뉴에서 아래 중 하나를 입력해 `분석`하면 에피소드 목록이 표시됩니다.

- 프로그램 URL 예: `https://anikids.ebs.co.kr/anikids/program/show/BA0AA000000000020`
- 재생 페이지 URL 예: `https://anikids.ebs.co.kr/vodCommon/show?siteCd=AK&courseId=...`
- 코스 ID 예: `BA0AA000000000020`

시즌 목록이 비어 있거나 자동 선택이 실패하는 경우, `stepId(직접입력)`에 아래처럼 값을 넣고 다시 `분석`하세요.

- 예: `60057279`
- 예: `01BA0AC000000000175`

분석 후 `선택 다운로드 추가`를 누르면 해당 항목들이 다운로드 큐에 즉시 등록됩니다.

## 쿠키 파일 만들기(권장)

FlaskFarm이 도커/서비스 환경이라 서버에서 브라우저 쿠키를 직접 읽기 어렵다면, 아래 방식 중 하나로 쿠키 파일을 만들어 `쿠키 파일 경로`로 읽어오세요.

### 방법 1) 브라우저 확장 프로그램으로 `cookies.txt` 내보내기

- 브라우저에서 EBS/애니키즈에 로그인
- `cookies.txt`(Netscape 포맷)로 내보내기
- FlaskFarm이 접근 가능한 경로에 저장 후 플러그인에서 `쿠키 파일에서 불러오기`

### 방법 2) Windows에서 쿠키를 파일로 자동 저장(주기 갱신)

```powershell
# Windows에서 실행
pip install -U browser-cookie3
python C:\EBS_Plugin\plugin\ebs_downloader\tools\export_cookie_windows.py --browser chrome --out C:\EBS_Plugin\data\ebs_cookie.txt --interval 600
```

- 위 파일(`C:\\EBS_Plugin\\data\\ebs_cookie.txt`)을 FlaskFarm 컨테이너의 `{PATH_DATA}/ebs_cookie.txt` 같은 경로로 공유(볼륨 마운트 등)하면,
  - 플러그인에서 `쿠키 파일 경로`를 `{PATH_DATA}/ebs_cookie.txt`로 설정해 자동 갱신에 활용할 수 있습니다.

## 자동 쿠키 갱신 동작

- 쿠키가 비어 있거나, 다운로드 시 로그인 상태가 아니면 자동 갱신 시도
- 자동 갱신은 1) 저장된 계정(ID/PW) 2) 쿠키 파일 3) 브라우저 쿠키(가능한 환경) 순서로 시도
- 설정 화면의 `저장된 계정으로 쿠키 갱신` 버튼으로 수동 점검 가능

## 필터 규칙

- `다운로드 모드 = blacklist`:
  - 프로그램/에피소드 키워드와 일치하면 제외
- `다운로드 모드 = whitelist`:
  - 프로그램/에피소드 키워드 중 하나라도 일치해야 다운로드
- 키워드는 `쉼표` 또는 `줄바꿈`으로 구분

## 참고

- 자동 갱신은 저장된 계정/쿠키 파일/브라우저 쿠키 중 가능한 방법으로 시도됩니다.
- 계정 정보 저장이 부담되면 자동 갱신을 `Off`로 두고 쿠키를 수동 갱신해서 사용하세요(또는 쿠키 파일만 사용).
- 브라우저 쿠키 직접 읽기는 FlaskFarm 실행 환경에서 브라우저 프로필 접근이 가능한 경우에만 동작합니다.
- FlaskFarm이 도커/서비스 환경이라 브라우저 쿠키를 직접 읽기 어렵다면, PC에서 `cookies.txt`를 내보내 공유 경로에 두고 `쿠키 파일 경로`로 읽어오는 방식을 권장합니다.
